This is an examproject created by Emil Jacobsen.

The code for the project is available in the RMD and HTML files.

The unaltered data i used can be found in the raw map, please note that to use it you wil have to convert the
encoding from ANSI to UGT-8.
this can be achieved through notepad++:
https://notepad-plus-plus.org/downloads/v8.4.8/
Open the file with notepad++ and go to the bottom right corner and change encoding from ANSI to UGT-8.
